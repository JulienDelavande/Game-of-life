\subsection{Detailed Description}
Fonctions about the simulation of the conways rules applied on the universe structure. 

\begin{Desc}
\item[Author:]Julien Delavande\end{Desc}
This are fonctions about the simulation of the conway game of of life applied on a universe structure. They are two different implementations, on considering the universe is a torus \mbox{[}torus implementation\mbox{]}. They can be recognize with \_\-torus word. The other implementation considers that the cell outside of the universe are dead and can't born \mbox{[}outside-dead implementation\mbox{]}. The implementation wich will be used in further files is the torus one.

The API of the \hyperlink{naive__conway_8h}{naive\_\-conway.h} is defined as follows:

\begin{itemize}
\item a function that apply the Conway's rules once on a given universe \mbox{[}outside-dead implementation\mbox{]}\item a function that apply the Conway's rules the number of steps of the universe \mbox{[}outside-dead implementation\mbox{]}\item a function that apply the Conway's rules once on a given universe \mbox{[}torus implementation\mbox{]}\item a function that apply the Conway's rules the number of steps of the universe \mbox{[}torus implementation\mbox{]}\item a function to test if a cell is alive\item a function to compte the number of cells alive around a given cell \mbox{[}outside-dead implementation\mbox{]}\item a function to compte the number of cells alive around a given cell \mbox{[}torus implementation\mbox{]} \end{itemize}
