\subsection{Detailed Description}
Simple definition of the structure Uni with fonctions. 

\begin{Desc}
\item[Author:]Julien Delavande\end{Desc}
This is a definition of the structure Uni which represent a universe. It contains the size of the universe an a matrix filled with char to represent the universe.

The API of the naive\_\-univer.h is defined as follows:

\begin{itemize}
\item a function to get the value of the universe's matrix\item a function to set the universe matrix's cells dead\item a function to display the universe in the terminal\item a function to switch the value in its other state of a cells in the universe matrix\item a function to set a cell's universe matrix alive\item a function to set a cell's universe matrix dead\item a function to create a copy of a given universe\item a fonction to desalocate the universe matrix \end{itemize}
